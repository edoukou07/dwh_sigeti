

CONFIGURATION CDC

	Instructions d'utilisation
	Configuration initiale :
	0-Exécutez d'abord set_stack_depth.sql dans psql ou pgAdmin
	1-Exécutez cdc_setup_source.sql dans sigeti_node_db
	2-Exécutez cdc_apply.sql dans sigeti_dwh (la première fois pour créer la structure)
	Chargement initial :

	3-Utilisez le script load_to_staging.sql existant pour copier toutes les données
	Après le chargement initial, mettez à jour cdc_state avec les IDs actuels si nécessaire
	Application périodique des changements :

	4-Exécutez SELECT apply_cdc(); dans sigeti_dwh régulièrement (via cron, tâche planifiée, etc.)
